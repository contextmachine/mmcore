# mmcore


## Base Docker Image

Pull image
```bash
docker pull ghcr.io/contextmachine/mmcore:latest
```
Using in Dockerfile
```dockerfile
FROM ghcr.io/contextmachine/mmcore:latest
```
## Install with pip
```bash
python -m pip install git+https://github.com/contextmachine/mmcore.git
```
